## Installation guide 

Create a virtual environment 

```
virtualenv venv
```

Start the virtual environment 

```
venv\Scripts\activate
```

Create a requirements.txt file and add these packages into those 

```
certifi==2022.12.7
charset-normalizer==3.0.1
idna==3.4
Naked==0.1.32
numpy==1.24.1
opencv-contrib-python==4.7.0.68
Pillow==9.4.0
psutil==5.9.4
PyYAML==6.0
requests==2.28.2
shellescape==3.8.1
urllib3==1.26.14
watchdog==2.2.1
```
